# Student Manager - WordPress Plugin

Đây là bài tập tạo Plugin quản lý Sinh viên.

## Chức năng
- Custom Post Type `Sinh viên`
- Meta Boxes (MSSV, Lớp, Ngày sinh)
- Shortcode `[danh_sach_sinh_vien]`
